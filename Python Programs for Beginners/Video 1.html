Simple Calculator

Statement: Make a simple Calculator using Python Programming which can perform Addition, Subtraction, Multiplication and devision.

 Note: No Python Packages should be used!!!

# Programme Topic: Simple Calculator

num_1 = int(input("Number 1\n"))
num_2 = int(input("Number 2\n"))
ope = input("Operator\n")

if ope == "+":
    result = num_1 + num_2
elif ope == "-":
    result = num_1 - num_2
elif ope == "*":
    result = num_1 * num_2
elif ope == "/":
    result = num_1 / num_2
else:
    print("Error: Operator Not Found!")

print(f"Result: {result}")
