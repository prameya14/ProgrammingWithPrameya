Python Project 5
Making Python Speak

Statement: We have to write a program which can make Python speak. We will break the code into a function so that we can call it whenever and wherever we want to.

 Note: We will use pyttsx3 module for this Project

To download source code and other material click on the button given below!

Source Code shown in the video 

import pyttsx3 # --> pip install pyttsx3

def speak(audio): # --> Our main function
    engine = pyttsx3.init("sapi5")
    voices = engine.getProperty("voices")
    engine.setProperty("voice", voices[0].id) # --> Male Voice for Windows
    # engine.setProperty("voice", voices[1].id) # --> Female Voice for Windows
    engine.say(audio)
    engine.runAndWait()

string = input("Enter what do you want to be told py Python\n")
speak(string)
