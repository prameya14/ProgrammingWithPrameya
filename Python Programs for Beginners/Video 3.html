Python Project 3
Guess Game

Statement: Make a guess game using Python. The user has to enter a number. If it is the correct guess then a message will be displayed. Chances must be hard coded in a variable. At last you have to display that how many chances did the user take to guess the correct number.
Hint: Random module is used!

 Note: Python Packages can be used!

# Programme Topic: Guess Game

import random

rand_num = random.randint(1, 50) # --> The random number will be between 1 and 50

chances_taken = 0
chances = 10
i = 1

while i <= chances:
    try:
        number = int(input("Enter your guess\n"))
    except:
        print("Error!")
    chances_taken += 1
    if number == rand_num:
        print("You got the correct guess!")
        print(f"Chances Taken to guess the correct number: {chances_taken}")
        break
    elif number != rand_num:
        if number > rand_num:
            print("You got the wrong guess!")
            print("Enter a smaller number!")
        else:
            print("You got the wrong guess!")
            print("Enter a larger number!")
    else:
        print("ERROR")
    i += 1
